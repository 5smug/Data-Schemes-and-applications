# Basics:

- To start the files, start your XAMPP on your computer.
- Start Apache and MySQL, nothing else
- After it started, use this link: http://localhost/phpmyadmin
- Press new, create a database named twin_cities (make sure the collation is utf8mb4_general_ci)
- Press create after that


# How to start:

- Press import on the twin_cities database, chose the database file first and import it
- After that's important, import the other file (with the long name)
